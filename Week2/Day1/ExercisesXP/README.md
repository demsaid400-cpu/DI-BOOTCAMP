# 📊 Multi-Dataset Data Analysis Project (Kaggle)

## 🧠 Description

Ce projet analyse plusieurs datasets Kaggle afin d’étudier les relations entre :
- 💤 le sommeil
- 🧠 la santé mentale
- 💳 les approbations de crédit

L’objectif est de pratiquer :
- le chargement de données
- le nettoyage
- l’analyse exploratoire (EDA)
- la visualisation des données

---

## 📁 Datasets utilisés

- Sleep Patterns Dataset
- Mental Health Dataset
- Credit Approval Dataset

*(Les fichiers doivent être téléchargés depuis Kaggle ou importés via Google Drive)*

---

## ⚙️ Technologies utilisées

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 🚀 Étapes du projet

1. Importation des bibliothèques
2. Chargement des datasets
3. Nettoyage des données
4. Analyse exploratoire (EDA)
5. Visualisation
6. Interprétation des résultats

---

## ▶️ Comment exécuter

### Sur Google Colab :
1. Ouvrir le notebook
2. Importer les datasets
3. Exécuter toutes les cellules

### Avec Kaggle :
- Télécharger les datasets
- Les placer dans le dossier du notebook

- Compréhension des patterns de sommeil
