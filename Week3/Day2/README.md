Global Power Plant Database Analysis

Objective

This project analyzes the Global Power Plant Database using NumPy, Pandas, Matplotlib, Seaborn, and SciPy.

Key Findings

* Hydroelectric and gas power plants are among the most represented worldwide.
* Power plant capacity varies significantly depending on fuel type.
* Statistical testing indicates significant differences in average capacity between some fuel categories.
* The global energy mix has evolved considerably over time.
* Geographic analysis reveals strong regional clustering of power plants.
* Eigenvalue decomposition highlights the dominant sources of variation in plant characteristics.
